# Secure Data Hiding in Image Using Steganography

This project demonstrates how to hide and extract data securely in images using Python. 
It includes encrypt and decrypt functionality and a user-friendly interface.

## Features
- Encrypt text into images
- Decrypt hidden text from images
- Simple Python implementation
- Tkinter GUI (optional)

## How to Run
1. Clone this repository
2. Install dependencies: pip install -r requirements.txt
3. Run the Python scripts: encrypt.py / decrypt.py
